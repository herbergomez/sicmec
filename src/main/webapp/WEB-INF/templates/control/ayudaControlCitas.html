Sistema de control y monitoreo de pacientes con enfermedades crónicas SICMEC.

Este manual describe las distintas funcionalidades y procedimientos que se pueden desarrollar dentro del Sistema de control y monitoreo de pacientes con enfermedades crónicas SICMEC para el hospital San Juan de Dios de Santa Ana y pretende ser una guía detallada paso a paso que el usuario puede consultar al momento de tener alguna duda o de presentarse alguna eventualidad.

1. Busqueda de pacientes

  • Para buscar un paciente dento del sistema se debe ingresar en el menú principal de la aplicación a la opción Control de citas.
  • En la pantalla de control de citas colocar el cursor del ratón en la caja de texto para búsqueda de pacientes. (Por defecto el sistema mostrara la lista de pacientes registrados en una tabla paginada. )
  • Presionar el botón Agregar Nueva Patología, el sistema mostrará una ventana modal para describir la nueva patología.
  • Digitar el expediente, nombres, apellidos o documento de identidad y presionar el botón Buscar.
  • El sistema refrescara la tabla con los pacientes que respondan al criterio de búsqueda.
  • Seleccione el paciente dentro de la tabla al que se le registrara una nueva cita ( Esto se realiza haciendo un click en la fila que contiene el registro del paciente).
  • El sistema abrirá una nueva ventana que contendrá 3 pestañas, la primera referente al expediente del paciente, la segunda de control de citas y la tercera de graficos.

2. Registro de nueva cita

  • Para registrar una nueva cita se debe ingresar en el menú principal de la aplicación a la opción Control de citas.
  • En la pantalla de control de citas colocar el cursor del ratón en la caja de texto para búsqueda de pacientes. (Por defecto el sistema mostrara la lista de pacientes registrados en una tabla paginada. )
  • Presionar el botón Agregar Nueva Patología, el sistema mostrará una ventana modal para describir la nueva patología.
  • Digitar el expediente, nombres, apellidos o documento de identidad y presionar el botón Buscar.
  • El sistema refrescara la tabla con los pacientes que respondan al criterio de búsqueda.
  • Seleccione el paciente dentro de la tabla al que se le registrara una nueva cita ( Esto se realiza haciendo un click en la fila que contiene el registro del paciente).
  • El sistema abrirá una nueva ventana que contendrá 3 pestañas, la primera referente al expediente del paciente, la segunda de control de citas y la tercera de graficos.
  • Haz un click en la pestaña Control de citas el sistema presenta un histórico (si existe) de las ultimas 20 citas médicas registradas para este paciente.
  • Al lado se encontrara el formulario de agregación de nuevas citas.
Ingrese todos los campos obligatorios (*)
  • Diagnostico (*)
  • Peso (*)
  • Estatura (*)
  • Tratamiento (*)
  • Dosis (*)
  • Periodicidad (*)
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • El sistema mostrara una alerta con el resultado de la operación anterior.

3. Ver detalle de citas anteriores

  • Para ver el detalle de citas anteriores se debe ingresar en el menú principal de la aplicación a la opción Control de citas
  • Haz un click en la pestaña Control de citas el sistema presenta un histórico (si existe) de las ultimas 20 citas médicas registradas para este paciente.
  • Haz un click en título de la cita que deseas detallar. El sistema desplegara en forma de drop down el detalle de la misma

4. Imprimir citas históricas

  • Para imprimir el detalle de citas anteriores se debe ingresar en el menú principal de la aplicación a la opción Control de citas, luego se debe continuar con los siguientes pasos
  • Haz un click en la pestaña Control de citas el sistema presenta un histórico (si existe) de las ultimas 20 citas médicas registradas para este paciente.
  • Presiona el botón Imprimir Informe al lado de la cita escogida.
  • El sistema dependiendo del navegador del cliente mostrara una vista previa en formato PDF o abrirá una caja de dialogo de impresión.

5. Anexar el resultado de un examen a una cita médica

  • Para registrar el resultado de un examen a una cita médica se debe ir al meno principal principal de la aplicación a la opción Control de citas
  • Haz un click en la pestaña Control de citas el sistema presenta un histórico (si existe) de las ultimas 20 citas médicas registradas para este paciente.
  • Presiona el bot&ocuate;n Agregar Exámenes al lado de la cita médica a la cual se desea anexar resultados de exámenes. El sistema despliega una pantalla modal con el formulario de agregación de exámenes y la lista de los exámenes que esta cita ya tiene registrados.
Ingrese todos los campos obligatorios (*)
  • Tipo de examen (*): Exámenes ya definidos dentro del sistema.
  • Resultado (*): Las unidades dependen del tipo de examen.
  • Comentario.

6. Ver gráficos de progresión de salud

  • Para ver el detalle de citas anteriores se debe ingresar en el menú principal de la aplicación a la opción Control de citas
  • Haz un click en la pestaña Histórico el sistema presenta una lista de los exámenes y signos vitales que el sistema soporta.
  • Al presionar el botón Imprimir Grafico El sistema imprime el grafico en pantalla.
