Sistema de control y monitoreo de pacientes con enfermedades crónicas SICMEC.

Este manual describe las distintas funcionalidades y procedimientos que se pueden desarrollar dentro del Sistema de control y monitoreo de pacientes con enfermedades crónicas SICMEC para el hospital San Juan de Dios de Santa Ana y pretende ser una guía detallada paso a paso que el usuario puede consultar al momento de tener alguna duda o de presentarse alguna eventualidad.

1. MANEJO DE PATOLOGÍAS

Descripción breve

El manejo de patologías consiste en el mantenimiento de la información que tiene que ver con las enfermedades principales que el sistema soporta, Así como también las distintas clasificaciones en las cuales se desglosa cada enfermedad.

Precondiciones

Autenticarse dentro de la plataforma con un usuario de rol Administrador.

1.1 Agregar Patología

  • Se debe ingresar a la opción Acciones. Dar click en la opción Patologías
  • A la izquierda de la pantalla se mostrará la lista de patologías generales que el sistema soporta.
  • Presionar el botón Agregar Nueva Patología, el sistema mostrará una ventana modal para describir la nueva patología.
  • Ingrese todos los campos obligatorios (*).
  • Nombre de la patología (*).
  • Descripción de la patología (*).
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se guardó con éxito.

1.2 Buscar Patología

  • Del lado izquierdo de la pantalla se muestra la lista de patologías generales soportadas por el sistema, para poder filtrar dentro de la tabla se deben seguir estos pasos:
  • Colocarse en la caja de texto Buscar.
  • Ingresar nombre o descripción.
  • El sistema refresca la tabla para mostrar las patologías que coinciden con el criterio de búsqueda.
  • Dentro de la tabla se pueden apreciar pequeños controles sobre los títulos de las columnas para poder sortear u ordenar los registros.

1.3 Editar Patología

  • Ingresar a la opción Acciones, luego a Patologías en el menú del sistema. A la izquierda de la pantalla se mostrará la lista de patologías generales que el sistema soporta.
  • Presionar el botón de edición dentro del registro que se requiere editar, el sistema mostrará una ventana modal para editar la patología.
  • Modifique los campos necesarios (*).
  • Nombre de la patología (*).
  • Descripción de la patología (*).
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se modificó con éxito.

1.4 Agregar Clasificación de Patología

  • Ingresar a la opción Acciones, luego a Patologías en el menú del sistema. A la izquierda de la pantalla se mostrará la lista de patologías generales que el sistema soporta y a la derecha la lista de sub divisiones definidas para cada patología a la izquierda junto con su respectiva descripción.
  • Presionar el botón Agregar nueva clasificación, el sistema mostrará una ventana modal para describir la nueva clasificación.
  • Ingrese todos los campos obligatorios (*).
  • Nombre de la clasificación (*).
  • Descripción de la clasificación (*).
  • Patología de la clasificación (*).
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se guardó con éxito.

1.5 Buscar Clasificación de Patología

  • Del lado derecho de la pantalla se muestra la lista de clasificaciones de patologías soportadas por el sistema.
  • Colocarse en la caja de texto Buscar.
  • Ingresar nombre, descripción o patología a la cual pertenece.
  • El sistema refresca la tabla para mostrar los tipos de patologías que coinciden con el criterio de búsqueda.
  • Dentro de la tabla se pueden apreciar pequeños controles sobre los títulos de las columnas para poder sortear u ordenar los registros.

1.6 Editar Clasificación de Patología

  • Se debe ingresar a la opción Acciones, luego a Patologías en el menú del sistema. A la izquierda de la pantalla se mostrará la lista de patologías generales que el sistema soporta y a la derecha la lista de sub divisiones definidas para cada patología a la izquierda junto con su respectiva descripción.
  • Presionar el botón de edición dentro del registro que se requiere editar, el sistema mostrará una ventana modal para editar la clasificación.
  • Modifique los campos necesarios (*).
  • Nombre de la clasificación (*).
  • Descripción de la clasificación (*).
  • Patología de la clasificación (*).
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se modificó con éxito.

2. MANEJO DE PACIENTES

Descripción breve

El manejo de pacientes se refiere al mantenimiento de toda la información de los pacientes que serán parte del sistema, aquellos con enfermedades crónicas.

Precondiciones

Autenticarse dentro de la plataforma con un usuario de rol Administrador.

2.1 Agregar Nuevo Paciente

  • Se debe ingresar a la opción Pacientes en el menú del sistema. El sistema mostrará la lista de pacientes registrados dentro del sistema.
  • El sistema mostrará la lista de pacientes registrados dentro del sistema.
  • Presionar el botón Agregar nuevo paciente, el sistema mostrará una pantalla modal con el formulario de agregación de nuevo paciente.
  • Ingrese los datos del paciente. Los campos (*) son requeridos.
  • Número de expediente (*).
  • Estado del paciente (*): Activo se refiere a un paciente con estado normal, Muerto es un paciente que ha fallecido, Inactivo es aquel paciente que tiene más de 6 meses de no retirar su medicamento y Abandono es aquel paciente que tiene 3 meses de no retirar su medicamento.
  • Patología (*): Se refiere a la enfermedad que padece este paciente.
  • Fecha de registro.
  • Nombres (*).
  • Apellidos (*).
  • Documento de Identidad.
  • Correo electrónico.
  • Fecha de nacimiento (*).
  • Sexo (*).
  • País (*).
  • Departamento (*).
  • Municipio (*).
  • Teléfono.
  • Dirección.
  • Información de contacto.
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se guardó con éxito.

2.2 Buscar Paciente

  • Se debe ingresar a la opción Pacientes en el menú del sistema. El sistema mostrará la lista de pacientes registrados dentro del sistema. Luego deberá continuar con los siguientes pasos:
  • Seleccionar en la lista desplegable el campo por el cual se realizará la búsqueda.
  • Colocarse en la caja de texto Buscar.
  • Ingresar nombre, apellido, DUI o expediente según opción seleccionada en lista desplegable.
  • El sistema refresca la tabla para mostrar los pacientes que coinciden con el criterio de búsqueda.
  • Dentro de la tabla se pueden apreciar pequeños controles sobre los títulos de las columnas para poder sortear los registros.

2.3 Editar Paciente

  • Se debe ingresar a la opción Pacientes en el menú del sistema. El sistema mostrará la lista de pacientes registrados dentro del sistema.
  • Identificar el paciente que se quiere editar y luego presionar el botón de edición , luego de esto el sistema despliega una pantalla modal con los datos del paciente.
  • Modifique los datos del paciente que usted desea. Los campos (*) son requeridos.
  • Número de expediente (*).
  • Estado del paciente (*): Activo se refiere a un paciente con estado normal, Muerto es un paciente que ha fallecido, Inactivo es aquel paciente que tiene más de 6 meses de no retirar su medicamento y Abandono es aquel paciente que tiene 3 meses de no retirar su medicamento.
  • Patología (*): Se refiere a la enfermedad que padece este paciente.
  • Fecha de registro.
  • Nombres (*).
  • Apellidos (*).
  • Documento de Identidad.
  • Correo electrónico.
  • Fecha de nacimiento (*).
  • Sexo (*).
  • País (*).
  • Departamento (*).
  • Municipio (*).
  • Teléfono.
  • Dirección.
  • Información de contacto.
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se modificó con éxito.

2.3 Ver Detalle de Paciente

  • Se debe ingresar a la opción Pacientes en el menú del sistema. El sistema mostrará la lista de pacientes registrados dentro del sistema.
  • Dentro del sistema identifique el paciente del cual requiere información adicional.
  • Presione el botón Ver detalle .
  • El sistema abrirá una pestaña en la que se podrá ver el detalle de toda la información del paciente, además de poder imprimir dicho detalle.

4. MANEJO DE MEDICAMENTOS

Descripción breve

El manejo de los medicamentos se refiere al mantenimiento todos los fármacos que el sistema soporta para poder ser agrupados y entregados a los pacientes con enfermedades crónicas.

Precondiciones

Autenticarse dentro de la plataforma con un usuario de rol Administrador.

4.1 Agregar Nuevo Medicamento

  • Se debe ingresar a la opción Medicamentos en el menú del sistema y luego presionar la sub opción Manejo de medicamentos. El sistema mostrará la lista de medicamentos registrados dentro del sistema en sus distintas presentaciones.
  • Presionar el botón Agregar nuevo medicamento, el sistema desplegará una pantalla modal con el formulario de agregación de medicamentos.
  • Ingrese los datos del medicamento. Los campos (*) son requeridos.
  • Nombre (*).
  • Descripción / Presentación (*).
  • Tipo: Institucional (Comprado por el MINED) o No institucional (Privado).
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrará mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenará los datos y mostrará una alerta de confirmación indicando si el registro se agregó con éxito.

4.2 Buscar Medicamento

  • Se presenta la lista de medicamentos dentro del sistema.
  • Colocarse en la caja de texto Buscar.
  • Ingresar nombre, descripción, estado o tipo.
  • El sistema refresca la tabla para mostrar los medicamentos que coinciden con el criterio de búsqueda.
  • Dentro de la tabla se pueden apreciar pequeños controles sobre los títulos de las columnas para poder sortear los registros.

4.3 Editar Medicamento

  • Identificar dentro de la tabla de medicamentos aquel fármaco que se necesita modificar.
  • Presione el botón de Edición al lado del medicamento a modificar, el sistema mostrara una ventana modal con la información del medicamento.
  • Una vez en el modal puede modificar los siguientes datos, los marcados con un asterisco son obligatorios:
  • Nombre (*).
  • Descripción / Presentación (*).
  • Tipo: Institucional (Comprado por el MINED) o No institucional (Privado).
  • Luego al modificar los campos necesarios se procede a presionar el botón Guardar.
  • El sistema almacenara los datos y mostrara una alerta de confirmación indicando si el registro se modificó con éxito.

5. MANEJO DE PAQUETES DE MEDICAMENTOS

Descripción breve

El manejo de los paquetes de medicamentos se refiere al mantenimiento todas las agrupaciones de fármacos que el sistema soporta para poder ser entregados a los pacientes crónicos.

Precondiciones

Autenticarse dentro de la plataforma con un usuario de rol Administrador.

5.1 Agregar Nuevo Paquete de Medicamentos

  • Se debe ingresar a la opción Medicamentos en el menú del sistema y luego presionar la sub opción Manejo de paquetes médicos. El sistema mostrará una lista de paquetes de medicamento previamente registrados dentro del sistema.
  • Presiona el botón Agregar nuevo paquete, el sistema mostrará una pantalla modal en la cual se deberá agregar los siguientes campos, aquellos marcados con (*) son obligatorios.
  • Nombre del paquete (*).
  • Descripción del paquete (*).
  • Luego de agregar esta información se debe presionar el botón Guardar.
  • Si no se han llenado todos los campos obligatorios el sistema no permitirá el guardado y mostrara mensajes de alerta.
  • Al ingresar todos los campos obligatorios y presionar el botón Guardar el sistema almacenara los datos y mostrará una alerta de confirmación indicando si el registro se modificó con éxito.

5.2 Editar Paquete de Medicamentos

Para realizar esta acción se deben realizar los siguientes pasos:

  • Seleccionar dentro de la lista el paquete médico que se quiere editar.
  • Presionar el botón Modificar paquete, el sistema desplegará una pantalla modal con la información del paquete médico.
  • Modificar la información del paquete.
  • Presionar el botón Guardar el sistema mostrará un mensaje de confirmación indicando el resultado de la operación.

5.3 Agregar/Remover Medicamento de Paquete

  • Ingresar a la opción Medicamentos en el menú del sistema y luego presionar la sub opción Manejo de paquetes médicos. El sistema mostrará una lista de paquetes de medicamento previamente registrados dentro del sistema, al seleccionar uno de ellos dentro de la lista, el sistema mostrará en el cuadro inferior izquierdo la lista de medicamentos que pertenecen a este paquete y en el cuadro inferior derecho la lista de medicamentos disponibles para este paquete.
  • Seleccionar en la lista un paquete médico.
  • Seleccionar en los cuadros inferiores el medicamento que se quiere agregar o eliminar del paquete y presionar el botón Eliminar del paquete ó Agregar al paquete dependiendo de la operación.
  • Al terminar el proceso presione el botón Guardar Cambios en la parte inferior de la pantalla, el sistema mostrará un mensaje de confirmación indicando el resultado de la operación.

6. AUDITORíA DE SISTEMA

Descripción breve

Se refiere al registro y posterior análisis de las acciones de los usuarios dentro del sistema.

Precondiciones

Autenticarse dentro de la plataforma con un usuario de rol Administrador.

6.1 CONSULTA DE AUDITORÍA

  • Ingresar a la opción Acciones en el menú del sistema y luego presionar la sub opción Auditoria. El sistema mostrará la lista de usuarios registrados dentro del sistema, seguido de dos campos con formato de fecha y hora para realizar las consultas
  • Seleccione un usuario dentro de la lista.
  • Coloque los rangos de fecha para los cuales se necesita realizar la consulta de acciones.
  • Presione el botón Buscar.
  • El sistema refrescara la tabla en la parte inferior con el detalle de auditoria de acciones para el usuario seleccionado en el rango de fecha requerido.
  • En caso de no encontrarse ningún registro de actividad, la tabla mostrara el mensaje "Ningún dato disponible en esta tabla".

7. REPORTES

Descripción breve

Muestra los diferentes reportes que fasicilitan el analisis al personal encargado para ello.

Precondiciones

Autenticarse dentro de la plataforma con un usuario de rol Administrador.

7.1 Verificar Reporte

  • Se debe ingresar a la opción Acciones en el menú del sistema y luego presionar la sub opción Reportes. El sistema mostrará la lista de reportes soportados por el sistema
  • Identificar el reporte que se quiere consultar.
  • Presionar el botón Ver al lado del reporte.
  • En el caso de los reportes que no requieran parámetros el sistema los mostrara en formato PDF en una pestaña diferente en el navegador.
  • Para aquellos reportes que se requieran parámetros como limites fechas, departamentos o municipios el sistema mostrara una pantalla extra en una pestaña del navegador.
  • Presiona el botón Generar Reporte.
  • El sistema mostrará el reporte en formato PDF.
