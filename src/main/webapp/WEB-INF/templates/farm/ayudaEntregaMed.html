Sistema de control y monitoreo de pacientes con enfermedades crónicas SICMEC.

Este manual describe las distintas funcionalidades y procedimientos que se pueden desarrollar dentro del Sistema de control y monitoreo de pacientes con enfermedades crónicas SICMEC para el hospital San Juan de Dios de Santa Ana y pretende ser una guía detallada paso a paso que el usuario puede consultar al momento de tener alguna duda o de presentarse alguna eventualidad.

1. Busqueda/Detalle de paciente

  • Para buscar un paciente dento del sistema se debe ingresar en el menú principal de la aplicación a la opción Entrega de Medicamentos. El sistema mostrará la pantalla principal de entrega de medicamentos.
  • Colocar el cursor del ratón en la caja de texto para búsqueda de pacientes.
  • Colocar el Número de Expediente y dar click al boton Buscar. El sistema mostrará el detalle de tratamiento que resetó el doctor que atendió al paciente asi como tambien puede verse el historial de entregas de medicamento.
  • Puede ver la información del paciente dando click al botón Detalle paciente. El sistema mostrará la información respectiva.

2. Entrega de Medicamento

  • Realizar una entrega de medicamento requiere primeramente buscar un paciente mediante su número de expediente. Vea Buscar un paciente/Ver expediente del paciente.
  • Dar click al botón Realizar Entrega. El sistema mostrará una ventana en la que se encuentra el detalle del tratamiento, dosis, paquete de medicamento. Dar click al botón Guardar
  • El sistema registrará la entrega y mostrará en la parte izquierda de la pantalla la lista de entregas hechas para dicho paciente.
  • Si se requiere realizar una entrega de medicamento antes de la sigueinte entrega, entonces será necesario realizar tal acción bajo aprobación, para ello el sistema mostrará una ventana en la que debe de ingresar el usuario y contrase√±a asi como tambien el motivo por el cual se esta realizando una enrega antes del tiempo estipulado. En caso que el sistema le muestre el mensaje Usuario o contraseña incorrecta es necesario comunicarse con el Administrador de Farmacia.

3. Historial/Detalle de Entregas

  • Para ver el historial de entregas realizadas a un paciente primero se debe de buscar dento del sistema para ello se debe ingresar en el menú principal de la aplicación a la opción Entrega de Medicamentos. El sistema mostrará la pantalla principal de entrega de medicamentos. Busque el paciente. Vea Buscar un paciente/Ver expediente del paciente.
  • El sistema mostrará en la parte izquierda de la pantalla la lista de entregas hechas para dicho paciente.
  • Hacer click en el botón para ver detalle de entrega.
  • El sistema mostrará la información de dicha entrega en una nueva ventana dando la opción de Imprimir dicho detalle.
